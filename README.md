## 项目简介 (你干嘛~哎哟！)
这是一款专门为小黑子们打造的ikun游戏！只要你会打篮球，会唱、跳、rap，这个游戏就是为你准备的！游戏融合了鸡哥的篮球元素和打砖块玩法，简直就是小黑子们的最爱！

## 技术栈 (鸡你实在是太美)
- Python 3.x (蔡徐坤同款编程语言，鸡哥用Python打篮球)
- Pygame (篮球音效必备，唱跳rap全靠它)
- Pillow (图片处理，让鸡哥更帅)
- OpenCV-Python (背景特效，让鸡哥转起来)
- NumPy (数值计算，算算鸡哥投篮命中率)
